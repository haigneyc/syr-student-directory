1. Website Creation & Technical Architecture
1.1 Site Positioning & Brand

Working Name Examples

CuseStudentDeals.com

OrangeDiscounts.com

SyracuseStudentSavings.com

Core Value Proposition

“Every legit student discount in Syracuse — verified, local, and updated.”

Tone:

Student-friendly, but trustworthy

Not spammy or coupon-cluttered

Clearly local, not generic

1.2 Platform & Stack (MVP)

Recommended Approach

WordPress + lightweight directory plugin
OR

Static frontend (Next.js / Astro) + Airtable / Supabase backend

Given your technical strength:

A static frontend + CMS-style backend is ideal long-term

MVP can start on WordPress for speed, then migrate later if needed

Core Pages

Home (featured deals + categories)

Category pages (Food, Retail, Entertainment, Services, Online)

Individual deal pages (SEO landing pages)

Submit a Deal (students + businesses)

About / Trust & Verification

Contact / Advertise

1.3 Deal Page Structure (SEO-Driven)

Each deal gets its own indexable page.

Example Page Structure

Title: “10% Off at XYZ Pizza – Syracuse University Student Discount”

Business name + logo

Discount details

Conditions (ID required, days, exclusions)

Location + map

Last verified date

Call-to-action (“Show student ID” or “Redeem online”)

This structure allows hundreds of long-tail SEO pages over time.

3. SEO & Traffic Strategy
3.1 Core SEO Thesis

This is a local + intent-driven SEO play, not a content mill.

You are targeting:

Students actively searching for savings

Parents helping students budget

New students onboarding

3.2 Primary Keyword Targets

Core Keywords

“Syracuse student discounts”

“Syracuse University student deals”

“Student discounts near Syracuse University”

Long-Tail Keywords

“student discount pizza syracuse”

“Syracuse student gym discount”

“SU student movie tickets”

Each deal page becomes a long-tail landing page.

3.3 On-Page SEO Strategy

Location-rich titles (“Syracuse”, “Syracuse University”)

Schema markup for LocalBusiness where applicable

Internal linking between categories and deals

Fast load times (important for mobile student traffic)

3.4 Off-Page & Distribution

Word-of-mouth via businesses themselves

Business Amplification

Give businesses a “Listed on Syracuse Student Deals” badge

Encourage them to link/share
